Graphite

Grahite is a interpreter written entirely in C++. The language uses a hybrid execution model. Instead of constructing a complete AST (Abstract Syntax Tree) it parses the program into a linear sequence of instructions. And whenever an instruction contains an expression a temporary AST is generated at runtime and after evaluation the tree gets discarded, only the value remains, thus keeping the interpreting linear. In 'theory' this should result in less memory usage while still handling operator precedence correctly. The language is still in its early stages of development.
